Instrucciones: Tarea corta 2

En este minijuego, se te emparejará aleatoria y anónimamente con otro participante. Uno de ustedes será el Participante A y el otro el Participante B. {{ if player.id_in_group == 2 }} Tú eres el Participante B. {{ endif }} {{ if player.id_in_group == 1 }} Tú eres el Participante A. {{ endif }}

Al inicio, el Participante A recibe {{ Constants.endowment }} puntos. El Participante B no recibe nada. El Participante A puede enviar un poco de o todos sus {{ Constants.endowment }} puntos al Participante B. Antes de que el Participante B reciba la cantidad, esta se multiplicará por {{ Constants.multiplier }}. Una vez que el Participante B recibe la cantidad triplicada puede decidir enviar un poco de o toda ella al Participante A.

En este minijuego, 50 puntos equivalen a S/1 (un sol).
