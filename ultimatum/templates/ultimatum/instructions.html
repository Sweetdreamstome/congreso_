Instrucciones: Tarea corta 3

En este minijuego, se te emparejará aleatoria y anónimamente con otro participante. Uno de ustedes será el Participante A y el otro el Participante B. {{ if player.id_in_group == 2 }} Tú eres el Participante B. {{ endif }} {{ if player.id_in_group == 1 }} Tú eres el Participante A. {{ endif }}

Existen {{ Constants.endowment }} puntos para dividirse. El Participante A decidirá con cuánto se quedará. Luego, el resto será ofrecido al Participante B. Entonces, el Participante B podrá aceptar o rechazar el ofrecimiento. Si el Participante B acepta, el monto será dividido según la propuesta. Si el Participante B rechaza el ofrecimiento, ambos participantes recibirán nada.

En este minijuego, 50 puntos equivalen a S/1 (un sol).
